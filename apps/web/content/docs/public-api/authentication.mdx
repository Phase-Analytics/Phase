Use a Public API token for every request. Public API tokens are separate from your SDK API key.

- **SDK API key** sends analytics data to Phase.
- **Public API token** reads analytics data from Phase.

## Get a token

<Steps>

<Step>

### Open API Keys

Open your app in the Phase dashboard and go to <strong>Application → API Keys</strong>.

</Step>

<Step>

### Create a Public API token

Create a token, choose an expiration, and copy it immediately.

Public API tokens are shown once and start with `phase_public_`.

</Step>

<Step>

### Send the token as a bearer token

<Tabs defaultValue="curl" groupId="public-api-auth" persist>
  <TabsList>
    <TabsTrigger value="curl">cURL</TabsTrigger>
    <TabsTrigger value="fetch">fetch</TabsTrigger>
  </TabsList>
  <TabsContent value="curl">
    <CodeBlock lang="bash" allowCopy={true}>
```bash
curl "https://api.phase.sh/public-api/v1/apps/<app_id>/reports/events/overview" \
  -H "Authorization: Bearer phase_public_your_token"
```
    </CodeBlock>
  </TabsContent>
  <TabsContent value="fetch">
    <CodeBlock lang="ts" allowCopy={true}>
```ts
const response = await fetch(
  'https://api.phase.sh/public-api/v1/apps/<app_id>/reports/events/overview',
  {
    headers: {
      Authorization: 'Bearer phase_public_your_token',
    },
  }
);

if (!response.ok) {
  throw new Error(`Request failed with ${response.status}`);
}

const data = await response.json();
console.log(data);
```
    </CodeBlock>

## Token lifetime

When you create a token in the dashboard, choose one of these expiration options:

- 1 week
- 1 month
- 1 year
- Never

You can revoke a token at any time from the dashboard. Revoked tokens stop working on the next request.

## Security guidance

<Callout type="warn">
Do not use Public API tokens in mobile apps, browser bundles, or any other untrusted client. Use them from your own backend, jobs, or internal tooling.
</Callout>

- Store tokens in environment variables or your secret manager.
- Rotate or revoke tokens if you suspect misuse.
- Copy the token when it is created. The plain token is not shown again.
- Phase stores only a hash of the token at rest.
