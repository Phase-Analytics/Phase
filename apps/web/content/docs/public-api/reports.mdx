## Events

### Overview

Use this endpoint for headline event counts.

<Tabs defaultValue="curl" groupId="public-api-events-overview" persist>
  <TabsList>
    <TabsTrigger value="curl">cURL</TabsTrigger>
    <TabsTrigger value="fetch">fetch</TabsTrigger>
  </TabsList>
  <TabsContent value="curl">
    <CodeBlock lang="bash" allowCopy={true}>
```bash
curl "https://api.phase.sh/public-api/v1/apps/<app_id>/reports/events/overview" \
  -H "Authorization: Bearer phase_public_your_token"
```
    </CodeBlock>
  </TabsContent>
  <TabsContent value="fetch">
    <CodeBlock lang="ts" allowCopy={true}>
```ts
const response = await fetch(
  'https://api.phase.sh/public-api/v1/apps/<app_id>/reports/events/overview',
  {
    headers: {
      Authorization: 'Bearer phase_public_your_token',
    },
  }
);

const data = await response.json();
```
    </CodeBlock>
  </TabsContent>
</Tabs>

### Timeseries

Use this endpoint for daily event counts over a date range.

<CodeBlock lang="bash" allowCopy={true}>
```bash
curl "https://api.phase.sh/public-api/v1/apps/<app_id>/reports/events/timeseries?startDate=2026-04-01T00:00:00.000Z&endDate=2026-04-06T23:59:59.000Z" \
  -H "Authorization: Bearer phase_public_your_token"
```
</CodeBlock>

### Breakdown

Supported dimensions:

- `eventName`
- `screenName`

<CodeBlock lang="bash" allowCopy={true}>
```bash
curl "https://api.phase.sh/public-api/v1/apps/<app_id>/reports/events/breakdown?dimension=eventName&limit=10" \
  -H "Authorization: Bearer phase_public_your_token"
```
</CodeBlock>

## Sessions

### Overview

Use this endpoint for total sessions, active sessions, average session duration, and bounce rate.

<CodeBlock lang="bash" allowCopy={true}>
```bash
curl "https://api.phase.sh/public-api/v1/apps/<app_id>/reports/sessions/overview" \
  -H "Authorization: Bearer phase_public_your_token"
```
</CodeBlock>

### Timeseries

Supported metrics:

- `sessionCount`
- `avgSessionDuration`
- `bounceRate`

<CodeBlock lang="bash" allowCopy={true}>
```bash
curl "https://api.phase.sh/public-api/v1/apps/<app_id>/reports/sessions/timeseries?metric=sessionCount&startDate=2026-04-01T00:00:00.000Z&endDate=2026-04-06T23:59:59.000Z" \
  -H "Authorization: Bearer phase_public_your_token"
```
</CodeBlock>

### Breakdown

Supported dimensions:

- `platform`
- `country`
- `city`

<CodeBlock lang="bash" allowCopy={true}>
```bash
curl "https://api.phase.sh/public-api/v1/apps/<app_id>/reports/sessions/breakdown?dimension=platform&limit=10" \
  -H "Authorization: Bearer phase_public_your_token"
```
</CodeBlock>

## Users

### Overview

Use this endpoint for total users, active users, new users, and location/platform summaries.

<CodeBlock lang="bash" allowCopy={true}>
```bash
curl "https://api.phase.sh/public-api/v1/apps/<app_id>/reports/users/overview" \
  -H "Authorization: Bearer phase_public_your_token"
```
</CodeBlock>

### Timeseries

Supported metrics:

- `activeUsers`
- `totalUsers`
- `newUsers`

<CodeBlock lang="bash" allowCopy={true}>
```bash
curl "https://api.phase.sh/public-api/v1/apps/<app_id>/reports/users/timeseries?metric=activeUsers&startDate=2026-04-01T00:00:00.000Z&endDate=2026-04-06T23:59:59.000Z" \
  -H "Authorization: Bearer phase_public_your_token"
```
</CodeBlock>

### Breakdown

Supported dimensions:

- `platform`
- `country`
- `city`

<CodeBlock lang="bash" allowCopy={true}>
```bash
curl "https://api.phase.sh/public-api/v1/apps/<app_id>/reports/users/breakdown?dimension=platform&limit=10" \
  -H "Authorization: Bearer phase_public_your_token"
```
</CodeBlock>
