Learn how to integrate Phase into your bare React Native application.

## Installation

Install the Phase SDK using your preferred package manager:

```bash
npm install Phase-react-native
# or
yarn add Phase-react-native
```

## Configuration

Configure your Phase instance with your API key:

```typescript
import { Phase } from 'Phase-react-native';

const Phase = new Phase({
  apiKey: 'your-api-key',
  endpoint: 'https://api.Phase.com',
});
```

## Tracking Events

Track custom events in your application:

```typescript
Phase.track('button_clicked', {
  buttonName: 'signup',
  page: 'homepage',
});
```
