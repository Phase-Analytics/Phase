Learn how to integrate Phase into your Expo application.

## Installation

Install the Phase SDK using your preferred package manager:

```bash
npx expo install Phase-expo
```

## Configuration

Configure your Phase instance with your API key:

```typescript
import { Phase } from 'Phase-expo';

const Phase = new Phase({
  apiKey: 'your-api-key',
  endpoint: 'https://api.Phase.com',
});
```

## Tracking Events

Track custom events in your application:

```typescript
Phase.track('button_clicked', {
  buttonName: 'signup',
  page: 'homepage',
});
```
