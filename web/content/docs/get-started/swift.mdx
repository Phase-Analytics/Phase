# Swift

Learn how to integrate Telemetra into your native iOS application.

## Installation

Add Telemetra to your project using Swift Package Manager:

```swift
dependencies: [
    .package(url: "https://github.com/telemetra/telemetra-swift.git", from: "1.0.0")
]
```

Or using CocoaPods:

```ruby
pod 'Telemetra', '~> 1.0'
```

## Configuration

Configure your Telemetra instance with your API key:

```swift
import Telemetra

let telemetra = Telemetra(
    apiKey: "your-api-key",
    endpoint: "https://api.telemetra.com"
)
```

## Tracking Events

Track custom events in your application:

```swift
telemetra.track("button_clicked", properties: [
    "buttonName": "signup",
    "page": "homepage"
])
```
