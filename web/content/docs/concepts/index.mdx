# Concepts

Learn about the core concepts and architecture of Telemetra.

## Events

Events are the fundamental building blocks of Telemetra. They represent user actions or system occurrences in your application.

## Sessions

A session represents a continuous period of user activity within your application.

## Devices

Telemetra tracks unique devices to help you understand your user base across different platforms.

## Real-time Analytics

Telemetra provides real-time analytics so you can monitor your application's performance as it happens.
